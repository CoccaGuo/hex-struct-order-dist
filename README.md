# hex-struct-order-dist

六角蜂窝结构的二维晶体中的有序无序相变。
程序描述了两种效应相互竞争的过程，在无序的体系中，晶化过程可能会有不同的pattern，两个可以调节的参数展示了它们如何控制不同图样的生成方式。
